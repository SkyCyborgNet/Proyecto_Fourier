## 📋 Descripción del Proyecto

Este proyecto implementa un análisis completo de señales utilizando la **Transformada de Fourier** en Python. El objetivo es visualizar y analizar señales en los dominios del **tiempo** y **frecuencia**, verificando las propiedades fundamentales de la transformada.

### 🎯 Objetivos

- Generar señales elementales (pulso rectangular, escalón, senoidal, etc.)
- Calcular la Transformada de Fourier de cada señal
- Visualizar espectros de magnitud y fase
- Verificar propiedades de la TF: linealidad, desplazamiento, escalamiento, modulación, dualidad y resolución espectral
- Crear una página web interactiva con los resultados
